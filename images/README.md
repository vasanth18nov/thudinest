# Property Photos

Add your Thudi Nest photos here. The site references these filenames:

- `exterior.jpg` — Outside view / front of the container home
- `bedroom.jpg` — Bedroom interior
- `bathroom.jpg` — Bathroom interior  
- `living.jpg` — Living area / kitchenette

**Tips:**
- Use landscape orientation (wider than tall) for best results
- Aim for 1200×800px or larger
- JPG format, keep files under 500KB for fast loading

Until photos are added, the page shows beautiful gradient placeholders automatically.
